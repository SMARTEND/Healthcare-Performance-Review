# Dashboard Screenshots

This folder contains PNG screenshots of all 4 pages of the Power BI dashboard (`healthcare_dashboard.pbix`).

| File | Dashboard Page |
|------|---------------|
| `01_consolidated_view.png` | Consolidated View — KPI hero cards, clinical/operational/financial metrics |
| `02_monthly_view.png` | Monthly View — time-series trends with YoY comparisons |
| `03_patient_details.png` | Patient Details — drill-through page filterable by cohort |
| `04_key_takeaways.png` | Key Takeaways — executive summary and recommended actions |

Screenshots are not auto-generated. To add them, open `powerbi/healthcare_dashboard.pbix` in Power BI Desktop, navigate to each page, and export via **File → Export → Export to PNG**.
